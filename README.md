# Student Management System

A modern web-based Student Management System built using React, TypeScript, and Vite. The application helps manage student information through an interactive dashboard and reusable UI components.

## Features

- Student dashboard
- Sidebar navigation
- Statistics cards
- Responsive design
- Reusable UI components
- Fast frontend with Vite

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation Guide

### Prerequisites

Make sure these are installed on your system:

* Node.js
* Git
* npm (comes with Node.js)

Check versions:

```bash
node -v
npm -v
git --version
```

---

# Install and Run Project

## Step 1: Clone Repository

Replace `your-username` with your GitHub username.

```bash
git clone https://github.com/your-username/student-management-system.git
```

---

## Step 2: Open Project Folder

```bash
cd student-management-system
```

---

## Step 3: Install Dependencies

```bash
npm install
```

This installs all required packages from `package.json`.

---

## Step 4: Start Development Server

```bash
npm run dev
```

---

## Step 5: Open in Browser

After running the command, terminal will show something like:

```bash
Local: http://localhost:5173/
```

Open that URL in browser.

---

# Build for Production

To create optimized production build:

```bash
npm run build
```

Generated build files will be inside:

```bash
dist/
```

---

# Preview Production Build

```bash
npm run preview
```

---

# Install VS Code Extensions (Recommended)

Install these extensions in Visual Studio Code:

* ES7+ React Snippets
* Tailwind CSS IntelliSense
* Prettier
* GitLens

---

# If npm install Gives Error

Run:

```bash
npm cache clean --force
npm install
```
